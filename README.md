# BudgetApp

A Swift-based iOS app for managing personal finances and tracking expenses.

## Features

- Import transactions from CSV and Excel files
- Automatic transaction categorization
- Visual spending analytics
- Smart expense tracking
- Category management
- Search functionality

## Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.0+

## Installation

1. Clone the repository
```bash
git clone https://github.com/YourUsername/BudgetApp.git
```

2. Open the project in Xcode
```bash
cd BudgetApp
open BudgetApp.xcodeproj
```

3. Build and run the project

## Usage

1. Launch the app
2. Import your transaction data (CSV or Excel)
3. View automatically categorized transactions
4. Analyze your spending patterns

## Testing

The app includes unit tests for the core functionality:

```bash
Command + U
```

## Architecture

- MVVM Architecture
- SwiftUI for UI
- Combine for data binding
- CoreXLSX for Excel file processing
